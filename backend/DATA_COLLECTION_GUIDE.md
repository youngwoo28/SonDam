# 수어 데이터 수집 가이드

## 목표
각 수어당 **50-100개 샘플** 수집하여 LSTM 모델 훈련

## 수집할 수어 목록
1. 안녕
2. 감사
3. 사랑
4. 도움
5. 괜찮다
6. 미안
7. 좋다
8. 나쁘다
9. 배고프다
10. 학생

## 수집 방법

### 1단계: 도구 접속
브라우저에서 접속: **http://localhost:8000/static/collect_data.html**

### 2단계: 수어 선택
- 드롭다운에서 수어 선택 (예: "안녕")
- 이름 입력 (선택사항)

### 3단계: 녹화
1. "📹 녹화 시작 (3초)" 버튼 클릭
2. 선택한 수어 동작 수행
3. 3초 자동 녹화 (약 90프레임)
4. 샘플이 자동 저장됨

### 4단계: 반복
- 같은 수어를 다양한 각도, 속도로 50-100회 반복
- 다양성이 중요:
  - 손 위치 변화
  - 동작 속도 변화
  - 카메라 거리 변화

### 5단계: 데이터 저장
- "💾 데이터 저장" 버튼 클릭
- JSON 파일 다운로드
- `backend/collected_data/` 폴더에 저장

## 데이터 품질 체크리스트

✅ 손이 명확하게 보임 (녹색 랜드마크 표시)
✅ 3초 동안 지속적으로 동작
✅ 배경이 깔끔함
✅ 조명이 충분함
✅ 한 번에 한 손만 사용

## 팁

### 효율적인 수집
- 한 수어당 10개씩 묶어서 저장
- 피로하지 않게 자주 휴식
- 동일한 동작이라도 약간씩 변화 주기

### 다양성 확보
- 앉아서 / 서서
- 빠르게 / 천천히
- 가까이 / 멀리
- 다양한 시작 위치

## 예상 소요 시간

- 수어 1개당: 약 10-15분 (50-100 샘플)
- 전체 10개: **2-3시간**

작은 배치로 나눠서 수집하는 것을 추천합니다:
- Session 1: 안녕, 감사, 사랑 (30-40분)
- Session 2: 도움, 괜찮다, 미안 (30-40분)
- Session 3: 좋다, 나쁘다, 배고프다, 학생 (40-50분)
